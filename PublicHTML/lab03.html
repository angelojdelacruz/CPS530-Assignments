Part A's task of using a fluid grid design with CSS was a lot tougher to complete. It was frustrating finding out the exact percentages and pixel measurements to get all the components to align nicely with each cell of the grid, as well as setting up the full grid itself. It also took much longer than using the Bootstrap framework and while I was happy with the final product, I think I much prefer using Bootstrap overall.

As you may tell with this site using the Bootstrap framework, I found it much easier to use and implement this. The only tough part of using Bootstrap is finding out how to use each class properly. Once I figured that out though, it was very smooth from there. Additionally, I like the look of the Bootstrap website a lot more than the one I made with CSS and fluid grids. It also took about half as much time to complete part B's task compared to part A.
