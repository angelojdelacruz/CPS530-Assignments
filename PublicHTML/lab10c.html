CPS530 Lab 10 Problem 3 - A page built in Metro 4

This webpage uses the Metro 4 framework. I installed it simply by importing its corresponding style sheet, and linking it's javascript file inside the page's html file.

While building this page, I repeatedly referred to the framework's documentation and how to set up certain parts like the above buttons to trigger these textboxes and aligning them to the center of the page. It felt similar to when I was learning how to use the Bootstrap framework, although this one feels both simpler to use and having more features to implement. I first tried having these boxes popup when you clicked on a button in a nav bar but found the drop-down buttons a lot easier to use.

The main difficulty that came with building this webpage was just figuring out the syntax and how to set up things like the drop down buttons and centering each element to look appealing. I solved these problems by looking at the documentation more closely and experimenting with the placing of each tag as well as different stylings.
